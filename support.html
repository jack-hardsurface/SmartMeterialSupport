Smart Material Generator - Support Page

Welcome to the official support page for Smart Material Generator. Here, you'll find information on how to install, use, and troubleshoot your addon.

📦 Installation Guide

1️⃣ **Download the addon** from Gumroad.

2️⃣ Open **Blender**, go to **Edit > Preferences > Add-ons**.

3️⃣ Click **Install**, select the downloaded ZIP file, and click **Enable**.

4️⃣ Done! 🎉 You’re ready to start using Smart Material Generator.

🛠 How to Use

• Navigate to the **Material Properties Panel** in Blender.

• Select your object and use the addon interface to generate smart materials automatically.

• Customize settings to achieve your desired material look.

🚀 Troubleshooting

• If the addon doesn’t install, restart Blender and try again.

• If you encounter errors, check **Window > Toggle System Console** for debug messages.
